Final Four Layout

Imprint newspaper

This was a 3 page feature piece in Imprint newspaper (See the web version here.) in which I helped with the layout. The story was written as a 4 part act, and it was important to the writer that the acts stayed intact.

The print and web version went on to win Imprint's Layout of the Term award. It also won the first place in Student Feature Writing for the 2014 BNC Awards.
