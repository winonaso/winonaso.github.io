Lead Me! A.R. Application

Course Work

This was an augmented reality application that I helped design for a user interface and user experience course at the University of Waterloo. Three-people teams created applications that used augmented reality functionality somehow. The application was a map application that helped students (particularly first year students) navigate around the campus in order to find different buildings, rooms, and services.

The design of this application spanned over four months, and we started at the very beginning by creating a persona and workflows. Part of the course included making paper prototypes and doing real user tests. The end result is a fully designed application, with real-world user tests and paper prototypes.
